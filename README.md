## Phonebook -App

This project, is build up by:
Frontend: ReactNative, Redux-toolkits, Tailwind CSS, Expo
Backend: Sanity

### Start Backend
`cd Sanity`
`npm install`
`sanity start`

Leave the server on, now run for Frontend

### Start Frontend
`npm install`
`npm start`
run depends on your emulator
